Documentation

1. Application created using OpenWeatherMap API
2. API source: https://openweathermap.org/
3. Options when using API: unit - city - temperature - weather icon - description - pressure - humidity
4. This weather API can be used to create data visualization for weather in an area or simply used as a searching tool for weather.


How to use Quick Weather

1. Enter a city name, followed with country code
2. Choose Farenheit or Celcius and view weather description in that city!
Enjoy :)
